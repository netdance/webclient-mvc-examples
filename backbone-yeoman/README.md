backbone-yeoman
======================

Example code from my explorations on setting up a Backbone app with yeoman, and following their best practice guidelines.

Includes:

* Backbone
* Bootstrap (SASS)
* RequireJS
* Grunt
* Bower
* Mocha
* jQuery
* Underscore
* Modernizr
* HTML5 boilerplate
* Support for CoffeeScript
* Support for Compass and SASS
